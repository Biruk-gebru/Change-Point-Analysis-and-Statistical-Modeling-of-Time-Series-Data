# Brent Oil Price Data

## Data Files

### Raw Data
- `brent_oil_prices.csv` - Historical Brent oil prices (May 1987 - Sep 2022)
  - Date: day-month-year format
  - Price: USD per barrel

### Events Data
- `geopolitical_events.csv` - Compiled list of major events affecting oil prices
  - Date: Event date
  - Event: Description
  - Category: Event type (political, economic, OPEC, conflict, etc.)
